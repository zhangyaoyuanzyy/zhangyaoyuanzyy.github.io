基本语法

  • 数据渲染：{{}}
    <p v-text="html1"></p>
    属性渲染：v-bind
    <h1 v-bind:title="message">hello</h1>
    <a :href="url">hello</a>
    修饰符：
    <input v-model.lazy="massage">
    <input v-model.number="massage">
    <input v-model.trim="message">
  • 类名渲染：
    <div v-bind:class="active : isActive"></div>
    <div v-bind:class="isActive ? active : red"> </div> 样式渲染：
    <div v-bind:style="{width:width}"></div>
    <div v-bind:sytle="styleObj"></div>
    <div v-bind:style="[styleObj1,styleObj2]"></div>
  • 条件绑定：
    <p v-if="seen">hahah</p>//有无内容
    <p v-show="seen">hah</p>//控制显隐
    循环绑定：
    <p v-for="list in lists">{{alist.text}}</p>
    <p v-for="(key value) in lists">{{key}}:{{value}}</p>
    <p v-for="(index key value) in lists">{{index}}:{{key}}:{{value}}</p>
  • 事件绑定：
    <a v-on:click="fun1">点击</a>
    <a @click="fun1">点击</a>
    事件修饰符：
    <a v-on:click.stop="doThis"></a>
    <form v-on:submit.prevent></form>
    <div v-on:click.capture="doThis">...</div>
  • 运算绑定：
    <p>{{reverseMess}}<p/>
    表单系列：
    <input type="checkbox" value="input1" v-model="message">
    <input type="radio" value="one" v-model="message">
    <select v-model="selected"></select>
  • <div id="vue10">{{reservemsg}}</div>

    reservemsg:function(){
    return this.message.split("").reverse().join("");
    }
    }
    <div id="vue10">{{reservemsg()}}</div>
    methods:{//方法属性
    reservemsg:function(){
    return this.message.split("").reverse().join("");
    }

组件间 通信

  • 场景：有两个组件A和B，A组件里面有个B组件，那么A就是B组件的父组件，B就是子组件。
    它俩的通信关系：父组件通过prop给子组件传递数据，子组件通过事件给父组件发送消息。

    Vue.component("child",{
          props:["message"],
          template:"<span>{{message}}</span>"
    })
    <child message="hello"></child>
    会渲染hello出来
    绑定整个数据对象：
    <div v-bind="todo"></div>
    todo:{
            text:"hahaha",
            age:10
    }
    用prop传递数据，在html上的规范
    <child my-msg="1"></child>//这个传递的是字符串1
    <child v-bind:my-msg="1"></child>//这个是传递的数字1，要用bind绑定起来
  • 动态绑定prop： <div id="vue20">
          <input type="text" v-model="message" name="">
          <child :msg="message"></child>
    //注意这里是绑定状态，不是直接的赋值，绑定了实例里的数据
    </div>
    var vue20 = new Vue({
            el:"#vue21",
            data:{
                    message:"hahahah"
            },
            components:{
                    "child":{
                            props:["msg"],//在子组件上声明的属性
                            template:"<p>{{msg}}</p>"
                    }
             }
    })
  • 单向数据流：就是父组件可以向子组件传递数据，传递的数据如果被修改，那么也会动态改变子组件里保存的数据，但是反过来就不行了。
    但是有两种情况：
    1、想在子组件里把传入的数据当做局部变量用。
    解决：props：["message"],
    data:function(){
            return {content :this.message}//把传入的数据赋值给其他变量
    }
    2、prop作为原始数据传入后，由子组件处理为其他数据输出
    解决：props:["size"],
            computed:{//定义一个计算属性，处理好prop的值后返回
                    normal:function(){
            return this.size.trim().toLowerCase();
        }
    }
